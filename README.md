# 🌐 Meus Sites Favoritos

Projeto desenvolvido como atividade prática de **HTML5** e **CSS3**, com o objetivo de criar uma página web simples, organizada e responsiva para apresentar uma lista com meus sites favoritos.

A proposta da atividade foi aplicar conhecimentos fundamentais de desenvolvimento web, utilizando uma estrutura semântica, estilização com CSS puro e organização visual por meio de cards.

---

## 📌 Sobre o Projeto

A página apresenta uma seleção de 5 sites favoritos, contendo para cada um deles:

- Nome do site;
- Imagem representativa;
- Link de acesso;
- Organização em formato de card;
- Layout responsivo para diferentes tamanhos de tela.

O projeto foi desenvolvido sem o uso de frameworks, bibliotecas externas ou JavaScript, utilizando apenas **HTML5** e **CSS3**.
